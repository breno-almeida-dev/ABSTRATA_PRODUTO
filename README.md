# Projeto Sistema de Gestão de Produtos
O sistema gerencia diferentes tipos de produtos, incluindo produtos alimentícios e de vestuário, permitindo operações como salvar, deletar e atualizar registros em um banco de dados relacional. O sistema também calcula o lucro com base no preço de custo e de venda dos produtos.


## 🚀 Começando

A implementação inclui classes para representar produtos, métodos para operações CRUD (criar, ler, atualizar e deletar) em um banco de dados, e um método para calcular o lucro dos produtos.
  

### Funcionalidades

- **Classes de Produtos:** Implementação da classe `Produto` e suas classes derivadas `ProdutoAlimenticio` e `ProdutoVestuario`.
- **Operações em Banco de Dados:** Métodos para salvar, deletar e atualizar registros de produtos em um banco de dados relacional.
- **Cálculo de Lucro:** Método `calcularLucro` que calcula o lucro com base no preço de custo e preço de venda do produto.
  

## 📋 Pré-requisitos

Para rodar o projeto, você precisará ter o seguinte software instalado:

- **Java JDK** (versão 8 ou superior)
- **IDE Eclipse** ou outra IDE de sua preferência.
- **Banco de dados relacional** (SQL Server, MySQL, PostgreSQL, etc.).

🔧 Instalação
- Clone este repositório:
    git clone https://github.com/breno-almeida-dev/ABSTRATA_PRODUTO.git
  
- Abra o projeto em sua IDE (recomendado o Eclipse).
- Compile e execute o código diretamente na IDE
